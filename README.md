# arena-nerf

A collection of software to run arena-nerf games.

## controller

A single controller to coordinate station nodes, run games, and serve the API.

## node

Game station nodes to read sensors and collect target hits.

## game

Game Engines to run different game modes.

## sensor

Sensor connectors to Raspberry Pi for reading target hits.

## ui

A web frontend console to display game information and start/stop games.
